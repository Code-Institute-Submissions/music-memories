Please ensure that any image links you submit are either your own or a Creative Commons Image

When using a Creative Commons Image
  1. Type Artist Name into Search Box and press Search.
  2. Click on your selected Image.
  3. Under "Credit The Creator" select the HTML Tab.
  4. Copy the IMG Reference only- do not copy the style or alt options.
  5. Paste this into the Image Link Box on this page.
  6. Click the "Rich Text Option".
  7. Click the Image Name on this Tab.
  8. Copy the link address you are directed to.
  9. Paste this into the Image Attributon Box on this page.
